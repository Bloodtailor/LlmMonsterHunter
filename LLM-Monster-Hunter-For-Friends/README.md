**The 15-second version:** It's a monster-catching RPG (think Pokémon)
where an AI makes up every monster, every battle, and every story beat
*as you play*. The game's code doesn't contain any monsters or plot — it
just manages the world and lets the AI do the imagining. Every creature is
one-of-a-kind, has its own personality, and remembers you.

Built solo, from scratch, as a learning project and portfolio piece.
